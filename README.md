### About

I build dependency-light, single-file web experiences — WebGL scenes, real-time AI tools wired to Groq's LPU inference, and interactive data visualizations — and ship them straight to static hosting. No backend gets bolted on unless the problem genuinely needs one.

B.E. Computer Science (2025) · IEEE published author · AWS Certified · IET member · national-level project expo award · prior internships in Java development and cybersecurity.
